Privacy Policy for LandlordPro

1. About LandlordPro

LandlordPro provides users (landlords) with tools to manage rental agreements. The user themselves registers and processes personal data, and LandlordPro only functions as a technical platform.

2. Who is responsible for the data?

  • The user (landlord) is the data controller for the personal data they register.
  • LandlordPro is a data processor and stores information on behalf of the user.

3. What data is processed?

The user can register the following information about tenants:

  • Name, address, phone number, and email of both the user and tenants
  • Rental agreements and payment history
  • Any expenses related to the rental agreement
  • Uploaded contracts and documents

4. How is this data used?

Personal data is registered and used only for managing the rental agreement.

5. Deletion and storage of data

  • The user can delete tenant data at any time.
  • The data is stored as long as necessary for management purposes.
